== Description ==
Entire Blocks is a new Gutenberg blocks to show the latest post in the WordPress page and post. It has color option, column settings, and more.


== Installation ==


= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Entire Blocks
3. Activate Entire Blocks plugin from your Plugins page.
4. Go to the Page and find Entire Blocks block.
5. Add in your page or post.

= From WordPress.org =

1. Download entire-blocks.
2. Upload the 'entire-blocks' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Entire Blocks from your Plugins page.


== Frequently Asked Questions ==
1. Where I can see the Settings?
Go to the WordPress Gutenberg page editor and find the blocks.
